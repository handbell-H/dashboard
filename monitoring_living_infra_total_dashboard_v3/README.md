# 생활인프라 종합 모니터링 대시보드 v2

v1 대비 대시보드 기능 3가지를 업데이트한 버전입니다.

---

## 프로젝트 구조

```
monitoring_living_infra_total_dashboard_v2/
├── dashboard.html        # 메인 대시보드 (빌드 후 생성)
├── build.py              # 데이터 → 대시보드 빌드 스크립트
└── README.md
```

데이터는 상위 폴더의 `../data/`를 참조합니다.

---

## v1 대비 변경사항

| 항목 | 내용 |
|------|------|
| 지도 - 시도 바로가기 버튼 | 지도 위에 시도 단위 버튼 추가. 클릭 시 해당 시도로 지도 줌 |
| 지도 - 지역 클릭 줌인 | 시군구 폴리곤 클릭 시 해당 지역 bounds로 자동 줌인 |
| 순위 비교 - 단계적 선택 | 비교 지역 선택을 시도 → 시군구 2단계 드롭다운으로 개선 |

---

## 빌드 방법

```bash
cd monitoring_living_infra_total_dashboard_v2
python build.py
```

출력: `dashboard.html` (브라우저에서 바로 열 수 있는 단일 파일)
